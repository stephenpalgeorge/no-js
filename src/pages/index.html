No JS

Building the web with as little JavaScript as possible

The No.js philosophy is to leverage the full power of HTML and CSS and build as much of the modern web as possible without using JS. This is as much out of curiosity as it is out of any philosophical convictions.

Browse the examples below my clicking on their name. You can also jump directly to the code by clicking the </> icon. If you've got any ideas for new components, or suggestions for the current implementations, please make a PR.

Caveats/Disclaimers

  • Please know, I am a passionate front-end developer by day and love JS.

    I'm not really campaigning to abandon JavaScript, I'm just saying that it's interesting to experiment with how much interactivity you can conjure up without it.

  • "No.js" really is only referring to the final code that is rendered in the browser.

    I'm fully aware that, in the source code, I'm using tools like Nunjucks - although this is, of course, optional. I also show these components implemented in popular JS frameworks (principally Svelte), but again it's really the output code that I'm interested in - how much JS is being shipped to the browser?
